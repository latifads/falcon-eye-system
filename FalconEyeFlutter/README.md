## Important

Flutter was not installed in the build environment where this was prepared, so the generated platform folders like `android/`, `ios/`, `macos/`, and `.dart_tool/` were not created here.

To finish turning this into a standard runnable Flutter project on your machine:

1. Install Flutter
2. Open a terminal in this folder
3. Run:

```bash
flutter create .
flutter pub get
flutter run
```

If Flutter asks whether to overwrite files, keep the existing:

- `lib/`
- `assets/`
- `pubspec.yaml`
